# Problem_Title

 - https://www.acmicpc.net/problem/4101
  

### 풀이 과정  
while문을 사용하여 조건이 a가 b보다 큰 경우는 YES를 출력, a,b가 0보다 작거나 같은 경우는 break하도록 하고 나머지 값은  
NO로 출력하도록 만들었습니다. 
while문을 빠저나가려고 할때 while문에 조건을 붙이는 것 보다 내부에 조건을 붙이는 것이 더 쉬울수도 있다는 것을 배웠습니다.

### 소스 코드

```c
#include <stdio.h>

int main(void){

    int a, b;

    while(1){
        scanf("%d %d", &a, &b);

        if (a > b)
            printf("Yes\n");

        else if(a!=0 && b!=0 && b >= a)
            printf("No\n");    

        else if(a==0 && b==0)  
            break;  

    }
    return 0;
}
```
